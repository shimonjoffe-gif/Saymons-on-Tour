# Saymons on Tour

Приложение для учёта фанатских выездов за Зенит.

## Запуск локально

### 1. База данных (Docker)
```
docker-compose up -d
```

### 2. Backend
```
cd backend
npm run db:generate
npm run db:migrate   # введите имя миграции: init
npm run dev
```
Сервер: http://localhost:3001

### 3. Frontend
```
cd frontend
npm run dev
```
Приложение: http://localhost:5173

## Структура

- **Выезды** — список матчей, переход к конкретному выезду
- **Участники** — управление семьями и участниками
- **Выезд** — отметка кто едет, расходы, итоговый расчёт долгов

## Логика расходов

| Категория | Взрослый | Ребёнок |
|-----------|----------|---------|
| Алкоголь | 1.0 | не участвует |
| Еда, транспорт, отель, экскурсия, прочее | 1.0 | 0.5 |

Веса можно изменить вручную при добавлении расхода.
